# Guías de Marca - Gestión SySO

Este documento establece los lineamientos visuales, conceptuales y de estilo para la interfaz de **Gestión SySO**, con el objetivo de lograr una identidad premium, coherente y profesional.

---

## 1. Identidad de la Marca
**Gestión SySO** es una plataforma SaaS B2B moderna que simplifica el cumplimiento de las normativas de Seguridad y Salud Ocupacional (SySO) y Medio Ambiente. 
- **Personalidad**: Confiable, profesional, moderna, segura, y de alta precisión.
- **Tono de Comunicación**: Claro, directo, preventivo y asertivo.

## 2. Paleta de Colores

Para transmitir seguridad industrial de una forma sofisticada y premium, se utiliza una paleta de colores basada en grises y negros profundos con acentos vibrantes inspirados en las señales de advertencia y seguridad de la industria.

### Colores Principales (Fondo y Estructura)
- **Primary Background (Slate 950)**: `#020617` (Un negro azulado muy elegante que sirve como lienzo de la UI en modo oscuro).
- **Secondary Background (Slate 900)**: `#0f172a` (Utilizado para tarjetas, páneles laterales y secciones flotantes).
- **Border / Muted Line (Slate 800)**: `#1e293b` (Líneas divisorias limpias y de bajo contraste).

### Colores de Estado y Acentos (Inspiración Industrial)
- **Acento Principal / Acción (Safety Orange)**: `#f97316` (Naranja brillante de seguridad industrial, usado para botones primarios, llamadas a la acción importantes y alertas críticas).
- **Advertencia / Atención (Warning Yellow)**: `#eab308` (Amarillo vibrante para estados intermedios, observaciones, riesgos menores o elementos pendientes).
- **Éxito / Aprobado (Safety Green)**: `#22c55e` (Verde claro para indicar que un área cumple con los estándares, auditorías aprobadas, y reportes limpios).
- **Información / Corporativo (Corporate Blue)**: `#3b82f6` (Azul para enlaces, guías de usuario o elementos neutrales de información).

## 3. Principios de Diseño Visual
1. **Glassmorphism Sutil**: Uso de efectos de desenfoque de fondo (`backdrop-filter: blur(8px)`) con bordes semitransparentes en tarjetas y barras de navegación superiores para dar sensación de profundidad.
2. **Gradientes Modernos**: Uso de degradados suaves para el fondo principal, botones o texto destacado (ej. un degradado de naranja de seguridad a amarillo de advertencia).
3. **Micro-animaciones**: Transiciones suaves (`transition: all 0.2s ease-in-out`) en botones, tarjetas clicables y campos de entrada.
4. **Espaciado Generoso**: Respetar el espacio en blanco para evitar la sobrecarga visual típica de los sistemas tradicionales de seguridad industrial.
